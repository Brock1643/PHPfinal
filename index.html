How to Easily Install a PHP-based Web Application

Installing a PHP-based web app, such as WordPress or Joomla, seems like it could be a difficult task. Following these steps can make it a little easier. I will be using WordPress as my example, but the installation process is similar with Joomla and other PHP-based web apps.

1.) Download App

Go to the website of the PHP-based web app you are looking for. For example, wordpress.org is where you would go for WordPress. The site is likely to have "download" buttons scattered on the page. As seen in the picture below, WordPress has one in the top right corner. You want to find a download button and download the application.


https://ecommerce-platforms.com/website-builder-reviews/wordpress-org-review



2.) Web Host

Now you will need a host for the website. Amazon Web Services has a version of this in AWS Lightsail. Create an account if you do not have one. Then, create an instance and select the Linux/Unix and LAMP Stack options.

https://levelup.gitconnected.com/what-is-amazon-lightsail-beaef47dd64e


3.) SFTP Host

You'll then need an application to do the File Transfer Protocol (FTP) for you. One example of this is Cyberduck. Upload the folder you downloaded off the PHP-based web app site. If you have the option to choose SFTP, do that to be secure. Connect the custom IP address to the SFTP host.

https://github.com/iterate-ch/cyberduck
